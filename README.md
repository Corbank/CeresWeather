# CeresWeather

A weather data processing and analysis project.

## Setup

1. Create a virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Project Structure

- `src/`: Source code
- `tests/`: Unit tests
